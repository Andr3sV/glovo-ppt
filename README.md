# Glovo Privacy & Compliance Presentation

Presentación interactiva de slides sobre Data Protection, Privacy y Compliance para Glovo.

## Tech Stack

- **Next.js 14** - Framework React
- **Tailwind CSS** - Estilos
- **TypeScript** - Tipado estático

## Colores Glovo

- `#F2CC38` - Amarillo (fondo principal)
- `#00a082` - Teal (acentos, headers)
- `#313131` - Gris oscuro (texto)

## Cómo ejecutar

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build producción
npm run build

# Iniciar producción
npm start
```

## Navegación

- **Flechas izquierda/derecha** o **espacio**: Navegar entre slides
- **Botones** en los laterales: Navegación visual
- **Barra de progreso** en la parte inferior

## Contenido

La presentación incluye 6 casos (scenarios):

1. **Global Data Protection Compliance Roadmap** - Timeline de 2 fases
2. **Privacy Roles & Group Structure** - Arquitectura de controladores
3. **Ex-Employee Data Incident** - Breach Response Playbook
4. **AI Support Chatbot** - Risk Assessment
5. **"Parrot" Feedback Platform** - Legal Memo
6. **In-App Advertising (EU & Morocco)** - Consent y compliance
